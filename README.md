# LowCostIoT
Hardware, Embedded software and Android App for Internet of Things, with low cost components.

Mapa do projeto
---------------
* **/TOM2015/Firmware** - Firmware e Código ESP8266 (Tomada e Interrptor)
* **/TOM2015/Hardware** - Arquivos de design (Eagle) da PCB, esquemáticos e gerbers (Tomada e Interrptor)
* **/TOM2015/App** - Código da aplicação Android
* **/TOM2015/Bin** - Arquivos binários (APK)
